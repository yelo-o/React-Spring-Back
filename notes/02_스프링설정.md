1. 명령 팔레트 메뉴에서 Spring Initializr 실행
2. Gradle Project 선택
3. Spring Boot 3.x 버전 선택 (JDK 17 이상 버전 사용하므로)
4. language : java
5. groupid : org.zerock
6. artifact Id : mallapi
7. packaging type : .jar
8. java version : 17
9. 의존성 라이브러리
    - Spring Boot Dev tools
    - Lombook
    - Spring Web
    - Spring Data JPA
    - MariaDB Driver
